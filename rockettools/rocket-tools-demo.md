# 데모 UI

## 프로젝트 배경

- 문제 탐색 단계에서 "마케터들이 CRM을 하기 위한 리소스가 없을 것" 이라는 가설을 세움
- 스모크 테스트를 진행했지만 랜딩 후 이탈률이 높음
- 로켓툴즈가 어떤 서비스인지 정확히 알 수 없다는 피드백을 받음
- 이를 해결하기 위해 데모 UI를 이용해 스모크 테스트를 진행해보기로 함
